# Todo App - CSC13008-GA03

A simple client-side rendering (CSR) todo application built with vanilla HTML, JavaScript, and TailwindCSS.

## Features

- **Client-side rendering** using vanilla JavaScript
- **Responsive design** with TailwindCSS
- **LocalStorage** for data persistence
- **Three pages**: Homepage, Todo List, Configuration

## How to Run

1. Open `index.html` in a web browser
2. No build process required (using TailwindCSS CDN)
